# Todo-App
#### Фронтенд взят с https://github.com/makedonsky-it/todo-app-frontend 

## Подготовка
Для работы приложения вам понадобится установленный `Docker`.

## Запуск приложения
При первом запуске, для создания образов контейнеров, необходимо ввести в терминале следующую команду:
  ```bash
  docker compose up --build
  ```
При последующих запусках можно вводить:
  ```bash
  docker compose up
  ```

## Отключение приложения
Необходимо остановить контейнеры дважды нажав `Ctrl + C` в терминале и ввести команду:
  ```bash
  docker compose down
  ```
Если необходимо удалить и все данные, хранящиеся в базе данных приложения, после предыдущей команды нужно ввести:
  ```bash
  docker volume rm todo-app_data
  ```
После чего ввести `y` в терминал для согласия на удаление.

#### Теперь приложение доступно по адресу http://localhost:3000/
