# Todo-App
#### Фронтенд взят с https://github.com/makedonsky-it/todo-app-frontend
## Подготовка
Для корректной работы приложения вам понадобятся установленные `Node.js` и `Docker`.

### Устанавливаем зависимости для `Node.js`
  1) Заходим в todo-app-fontend-master
     
  2) Открываем todo-app-frontend-master в терминале

  3) Вводим команду:  
     ```python
     npm install
     ```

После этого будут установлены все необходимые для Node.js зависимости 

### Поднимаем базу данных через `Docker`
Заходим в терминал и вводим следующие команды:
  1) Переходим в папку с нашим бэкендом:  
     ``` bash
     cd todo-app-backend
     ```
     
  2) Активируем виртуальное окружение:  
     Для Windows:
     ```bash
     venv\Scripts\activate
     ```
     
     Для Linux\macOS:
     ```bash
     source venv\bin\activate
     ```
     
  3) Создаём pg-контейнер:  
     ```bash
     docker run --name pg-container -e POSTGRES_PASSWORD=admin -d -p 15432:5432 postgres
     ```

Теперь в вашем Docker будет развернута база данных, в которой будут храниться все данные, которые вы передаёте в приложение.  
Вы также можете сворачивать и разворачивать её по своей воле.
На Linux и macOS возможно также развертывание через `colima`
#### !!!ВНИМАНИЕ!!!
В `POSTGRES_PASSWORD` будет храниться введённый вами пароль (в случае с командой выше пароль будет `admin`), его желательно запомнить.
Порт 15432 может быть занят на вашем устройстве. Вместо него можете написать любой другой незанятый.

## Запуск
### Поднимаем Фронтенд
Делаем следующие действия:

  1) Заходим в `todo-app-fontend-master`

  2) Открываем `todo-app-frontend-master в терминале

  3) Вводим команду:  
     ```python
     npm start
     ```
### Поднимаем Бэкенд
  1) Переходим в IDE в которой вы будете запускать код
    
  2) Открываем терминал IDE и переходим в нашу папку следующей командой:
     ```python
     cd todo-app-backend
     ```
  3) Активируем виртуальное окружение:
     Для Windows:
     ```bash
     venv\Scripts\activate
     ```
     Для Linux\macOS:
     ```bash
     source venv\bin\activate
     ```

  5) Поднимаем бэкенд:
     ```bash
     uvicorn app.main:app --port 8080 --reload
      ```
#### Теперь приложение доступно по адресу http://localhost:3000/
